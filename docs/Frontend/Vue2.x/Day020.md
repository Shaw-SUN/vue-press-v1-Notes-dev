## 表单

`@submit.prevent` 取消表单提交默认刷新页面

### 数据双向绑定

`v-model="数据名"`

#### 修饰符

- `.number` 自动转为 Number 类型(_如果原值的转换结果为 NaN 则返回原值_)
- `.trim` 自动过滤用户输入的首尾空格
- `.lazy` 将 v-model 在 input 事件中同步输入框的数据转变为在 change 事件中同步数据

### 数据交互

http 协议：前端（浏览器）发送请求，服务器给予响应。  
请求方法：get（查询）、post（添加）、put（修改）、delete（删除）  
ajax：不刷新页面与后台交互数据。  
axios：封装好的 aja× 模块。
